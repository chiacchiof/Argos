📖 Come funziona
  1. Aggiungi una chiave per ogni provider che vuoi usare (OpenAI, Anthropic, ...). Puoi averne piu' d'una per provider — tipica configurazione: "prod" e "dev".
  2. Il label e' il nome che vedrai nel dropdown del task form: scegli qualcosa di parlante ("OpenAI prod cliente X", "Claude personal").
  3. La chiave e' cifrata Fernet: nemmeno con accesso al DB si puo' leggerla senza ARGOS_SECRET.
  4. Per Ollama locale non serve API key — basta crearla (con campo chiave vuoto) per poterla selezionare nei task.
  5. In modifica, il campo chiave puo' restare vuoto = mantieni la chiave attuale. Scrivi CLEAR (maiuscolo) per cancellarla.
